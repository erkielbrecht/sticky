# Dependencies
  - `granite (>= 0.6.0)`
  - `libgtk-3-dev (>= 3.10)`
  - `python3 (>= 3.6)`
  - `python3-gi (3.36.0-1)`
  -  `gettext`
  -  `handy`
  -  `json`

# Building
  ```
    git clone https://github.com/ekrielbrecht/sticky.git
    cd sticky
    flatpak-builder build com.github.erkielbrecht.sticky.yml --user --install --force-clean
  ```
